# Danley Logistics — Images Required

Create an `images/` folder in the same directory as your HTML files and add the following images.
All images should be saved as .jpg or .png (update the src in HTML to match your file extension if needed).

- `danleylogo.png` — Company logo (any size, will be constrained to 36px height in nav)
